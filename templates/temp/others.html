1. Handwritten Digit Classification - link to demo video

Using CNN built on top of Tensorflow, i create the handwritten digit classifier, to classify handwritten digits from 0 to 9. I think there are tons of project like this out there. But, i modify the way the user interact with our model. Instead of uploading an image, user can directly draw their handwritten inside canvas object. I make 2 classifiers, the first one is built from scratch, while the other one use transfer learning . But due to dataset and computational power limitation the classifiers stil has many flaws.

Google News Scraper

Just a simple scraper to scrap news from news.google.com . I put certain condition such as : it only scrap hot news ( not more than 2 days ago), and other criteria (explained in web app). The news are scraped using selenium and beautifulsoup. It can receive multiple queries as long as you separate them by semi colon. Since the web app is simple, i deployed it using AWS EC2 free tier.
