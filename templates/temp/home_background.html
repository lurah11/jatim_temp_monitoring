Hi guys, this web application is intended as capstone project of the Tetris Batch 3 Program organized by DQ Lab. In this project i will try to apply my knowledge of ETL ( Extract, Transform, Load ) and data analysis in order to gain insight of temperature distribution in East Java using dataset obtained from BMKG. In order to fully understand this project, please kindly check all the sections in left sidebar in ordered fashion. :D But, if you just interested to see the data visualization and analysis, go ahead to Dashboard tabs. Meanwhile, if you are interested to view the implementation in code, please visit my github here

1. Background

The background of this research is actually not a something big. I am currently living in Mojoketo City (Kota Mojokerto) while my father lives in Malang Regency (Kabupaten Malang). Everytime i visit my father in Malang, the air feels so cold. Sometimes, i even got cold when staying several days in my father house

2. Research Question

  • i am interested to know whether the temperature of Malang Regency is actually significantly colder than temperature of Mojokerto City? or this is just "feels like" temperature and the temperature is actually not significantly different?
  • How about other cities? What is the average temperature distribution looks like?
  • While i am on it, can we expand the application to also visualize the humidity? Can we make this web app more flexible by receiving dynamic input from the users?
